# Technischer Datenschutz-Baustein für den Nur-notwendig-Modus

Dieser Text beschreibt ausschließlich das tatsächliche Verhalten des N24 Consent Managers ab Version 1.8.48, solange keine optionalen Statistik-, Marketing- oder externen Mediendienste aktiv sind. Er ersetzt keine rechtliche Prüfung der vollständigen Datenschutzerklärung.

Das Plugin fügt diesen Text nicht automatisch in WordPress ein und verändert keine Datenschutzseite. Der Baustein darf nur nach manueller Prüfung und bewusster redaktioneller Entscheidung übernommen werden.

## Formulierungsvorschlag

### Cookie-Informationen

Auf dieser Website wird der N24 Consent Manager eingesetzt, um Informationen zu technisch notwendigen Speichertechnologien bereitzustellen. Solange keine optionalen Dienste aktiviert sind, öffnet sich kein automatischer Einwilligungsbanner. Der Consent Manager speichert in diesem Modus keine Auswahl im Local Storage, setzt kein Consent-Cookie, erzeugt keine Consent-ID und führt keine Einwilligungshistorie. Es wird außerdem kein Einwilligungsprotokoll an den Server übermittelt. Die Cookie-Informationen können jederzeit über den entsprechenden Link im Footer geöffnet werden.

## Vor Verwendung prüfen

- Der Baustein gilt nur, wenn im Dialog tatsächlich keine optionalen Dienste aufgeführt sind.
- Werden später Analyse-, Marketing- oder externe Mediendienste aktiviert, muss der Text vor Veröffentlichung angepasst werden.
- Allgemeine Aussagen der Datenschutzerklärung zu Cookies, Reichweitenmessung oder „Optimierung des Webpublikums“ müssen mit den tatsächlich eingesetzten Diensten abgeglichen werden.
